Function for sending integers to the host.

This function is for sending regular integer numbers to the device the Arduino is connected to. This function is only to be used by other parts of the library. For sending data to the host from your own code, please use the Logging class.

Function for sending character arrays to the host.

This function is for sending an array with characters to the device the Arduino is connected to. The maximum size of this array is 50. This function is only to be used by other parts of the library. For sending data to the host from your own code, please use the Logging class.

Pauses the program.
